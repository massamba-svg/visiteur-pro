Comment ajouter une nouvelle visite ?

Cliquez sur le bouton « Ajouter une visite » dans le tableau de bord ou allez dans la section Visites. Remplissez le formulaire avec les informations requises (nom du visiteur, entreprise, personne rencontrée, motif de la visite et heure d'arrivée).

Comment enregistrer le départ d'un visiteur ?

Allez dans l'historique des visites récentes, trouvez le visiteur dont l'heure de départ n'est pas enregistrée, et cliquez sur le bouton « Enregistrer le Départ ».

Puis-je modifier une visite enregistrée ?

Oui, vous pouvez accéder à l'historique, sélectionner une visite, et utiliser le bouton d'édition pour la modifier. Cependant, les visites terminées ne peuvent être supprimées.

Ajouter un nouveau client

Accédez à la section Clients, cliquez sur « Ajouter un Client », puis remplissez les informations essentielles : nom, entreprise, email et téléphone.

Consulter l'historique d'un client

Sélectionnez un client dans la liste pour voir ses détails complets et l'historique de toutes ses visites avec les dates et les motifs.

Comment filtrer les clients ?

Utilisez la barre de recherche en haut de la section Clients pour filtrer par nom, entreprise ou email. Des options de tri sont également disponibles.

Comment créer un nouveau rapport ?

Allez dans la section Rapports, cliquez sur « Créer un Rapport », choisissez le type de rapport souhaité, puis suivez les étapes de configuration.

Puis-je personnaliser les rapports ?

Oui, vous pouvez sélectionner les données à inclure, choisir le format de présentation, et appliquer des filtres pour affiner les informations affichées.

Comment exporter un rapport ?

Après avoir généré un rapport, cliquez sur le bouton « Exporter » et choisissez le format souhaité (PDF, Excel, etc.).
